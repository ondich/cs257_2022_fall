CS 257: Sample Web App

This is a TERRIBLE user interface. Do not copy it for your project!

Why is this page this way? Because I want to give you an example that illustrates the core techniques you'll need for your web application project, but does so in as minimal a way as possible. So you'll find a little HTML, a little CSS, a little Javascript, and access to a database-drive API, but nothing more. You can't even search for a book or author by name, which would be fundamental features if I actually wanted to make this application useful.
