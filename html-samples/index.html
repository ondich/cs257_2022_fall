Introductory HTML/CSS samples

This page links to a few HTML/CSS samples to help you get started.

  • Get these files onto your computer. The easiest way to do that is to clone GitHub repository for this class and then look in the html-samples directory.
  • For each sample:
    • Use File->Open in your browser to open the sampleX.html file. (Alternatively, double-click on the HTML file, which should open the browser.
    • Right-click on the resulting browser page and select "View Page Source", which will show you the raw HTML in a new tab.
    • If there are CSS or Javascript files listed in the <head> section of the raw HTML, click on their links to get those files open in separate tabs.
    • Read the HTML. What do you expect it to do.
    • Think of things you'd like to try changing in these samples, then open the HTML and CSS files in your text editor. Play around with the HTML and/or the CSS and save your changes.
    • Reload the page in the browser. Generally, when you change the CSS, you have to "force-reload" to prevent your browser from just using its cached copy of the .css file. Sometimes you have to force-reload multiple times until the browser gets the hint. This is super-annoying. For most browsers, you can force-reload via Shift-Ctrl-R (Windows and Linux) or Shift-Command-R (Mac). If that doesn't work, check the internet for guidance.

  • Keep track of your questions and if you get stuck, ask in class or on Slack.

Here are the samples.

  • sample0.html: The minimal file structure for an HTML page. No CSS.
  • sample1.html: A demo of a bunch of basic HTML tags, including paragraphs, links, headings, images, lists, etc. No CSS.
  • sample2.html: Introduction to CSS styles applied to tags and to classes. Depends on sample2.css.
  • sample3.html: Longer introduction to CSS styles, with some exercise suggestions. Depends on sample3.css.
  • sample4.html: my little "navigation bar" sample, showing how styling can make substantial presentation changes to content like an unordered list of links. Depends on sample4.css.
  • sample5.html: layout with CSS. Depends on sample5.css.
  • sample6.html: layout with CSS Flexbox. Depends on sample6.css.

Resources for getting to know HTML and CSS

There is a ton of tutorial information online about HTML and CSS. Here are a few suggested starting points.

  • Obvious: search for "CSS [concept I'm looking for]" or "HTML 5 [concept]"
  • Excellent beginner and intermediate tutorials from HTML Dog
  • Mozilla Developer Network (MDN): very good tutorials, references, etc.
  • w3schools.com: very good tutorials, references, etc.
  • w3fools.com: used to say that w3schools was terrible and that everybody should use MDN instead, but now think w3schools has gotten better. Sounds like some public pressure may have helped w3schools fix its problems.
  • CSS tutorial from MDN
  • CSS tutorial from w3schools.com
  • Colors in CSS from w3schools.com
  • TheWorld Wide Web Consortium (W3C) is the official standards organization for lots of web-related protocols, languages, and tools, including HTML and CSS.
  • Watch out: W3C != w3schools.com. There's no formal connection between these two organizations, and sometimes the W3C people can be a little bit cranky about the confusion.
