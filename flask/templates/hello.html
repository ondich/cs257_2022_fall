An important attribute of the flask.render_template function is that it protects you against dangerous user input. For example, try the endpoint /hello/<em>Bob, which looks like it's intended by the user to turn everything from "Bob" to the end of the page italic. Check out what render_template produces instead. This is one way Flask provides you with tools to prevent injection attacks.
